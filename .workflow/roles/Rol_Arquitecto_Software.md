# Rol: Arquitecto de Software

## Misión Principal
Definir la estructura de alto nivel y las decisiones tecnológicas del proyecto WebChoroico. Su objetivo es garantizar que la aplicación sea escalable, segura y fácil de desplegar, tomando decisiones estratégicas sobre el stack tecnológico (Python/Flask) y la infraestructura (Hosting Compartido).

## Responsabilidades Clave
- **Definición de Arquitectura:** Diseñar la estructura de carpetas y la organización de módulos (ej. uso de Blueprints en Flask) para mantener el proyecto ordenado a medida que crece.
- **Compatibilidad con Hosting:** Asegurar que las decisiones técnicas sean compatibles con el entorno de Anaconda Web (Python en cPanel, limitaciones de permisos, versiones de librerías).
- **Selección de Tecnologías:** Evaluar y aprobar nuevas librerías, extensiones de Flask o herramientas de terceros necesarias para el proyecto.
- **Estrategia de Despliegue:** Definir y documentar el proceso de despliegue manual o semi-automatizado adecuado para cPanel.
- **Estándares Técnicos:** Colaborar en la evolución de los lineamientos de codificación (`guidelines/`) para asegurar que reflejan las mejores prácticas de la industria.
- **Rendimiento y Seguridad:** Supervisar que la arquitectura propuesta cumple con los requisitos no funcionales, optimizando el uso de recursos limitados del hosting compartido.
